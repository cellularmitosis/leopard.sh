This directory contains a few (statically-linked) linux ppc binaries:
- pv
- socat
- lzop

Additionally, this directory contains a "remastered" copy of the Finnix 111
PowerPC Linux Live CD which contains these additional binaries:
- finnix-ppc-111b.iso

"remaster.sh" is the script which was used to generate this iso image.

The original finnix-ppc-111.iso is available from:
- https://ftp.osuosl.org/pub/finnix/111/
